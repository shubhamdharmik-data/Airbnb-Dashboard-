# Airbnb Analytics Dashboard

This Power BI project analyzes Airbnb listing data to uncover insights related to property performance, ratings, and customer reviews.

Key Features: 

->Interactive Overview Dashboard
->Ratings Analysis
->Reviews Analysis
->KPI Tracking and Performance Metrics
->Data Modeling and DAX Measures
->User-friendly Visualizations

Tools Used:

->Power BI
->DAX
->Data Modeling
->Data Visualization

Objective:

To transform Airbnb data into actionable insights that help understand listing performance, customer satisfaction, and overall business trends.

Skills Demonstrated:

Power BI • DAX • Data Cleaning • Data Modeling • Data Visualization • Business Analytics

Project Type: End-to-End Data Analytics Dashboard Project 📈
